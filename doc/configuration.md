#### Config of Realtime-Database Rules

`
{
	"rules": {
     ".read": true,
     ".write": true,
     "recipes": {
       ".indexOn": ["receipt", "uid", "category", "rating"]
     },
     "users": {
			 ".indexOn": ["uid"]
     },
     "favorites": {
			 ".indexOn": ["uid"]
     }
   }
}
`
